# Introduction

An Angular 2 app to search runners by country.

# Installation
```bash
git clone https://github.com/hgkhanh/myNextRunTest.git
cd myNextRunTest
#install dependencies
npm install
#build and run 
npm run build.prod
npm run serve.prod
```
(Note: this project requires node and npm)
