# Introduction

An Angular 2 app to search runners by country.

# Installation
```bash
git clone https://github.com/hgkhanh/myNextRunTest.git
cd myNextRunTest
```
install the project's dependencies
```bash
npm install
```
run
```bash
# prod build
npm run build.prod
# Run production server
npm run serve.prod
```
(Note: this project requires node and npm)
